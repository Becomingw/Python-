# Python人民币纸币币值识别
基于计算机视觉的纸币币值识别
## :apple:基于[Paddle OCR](https://github.com/PaddlePaddle/PaddleOCR)
### :pear:需要的库：
本项目用的Python，需要安装opencv库与numpy还有Pillow库，具体安装方法自行bing。
